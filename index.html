Digital to analogic Conversion : What is that ?

Digital to analogic conversion is a process in which most of the tech world uses today.
It is used in many things, like a speaker which is an analogic output.
The computer is playing a song stored in a .mp3 format or something else which is digital information.
Then this information needs to be converted to analogic in order to the speaker to play.

And this is what we are going to see today !

Digital to analogic Conversion : How does it work ?

First, we will choose what is the digital value that we want to convert to analogic.
For example, Dval = 48
Then, we need to know what will be the voltage range we can work on.
For example, from 0 V to 5 V so, Vref = 5 V
We need to know what will be the DAC's resolution, let's say it is 8 bit.
With the resolution, (Rres = 8) we can find the Quantum :
Q = Vref / (2Rres - 1)
Q = 5 / (28 - 1) = 0.0196 V = 19.6 mV
